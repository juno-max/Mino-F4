**Date**: November 5, 2025

## ðŸŽ¯ Core Principles

### 1. Immediate Gratification
**After CSV upload, jobs should START EXECUTING automatically** with a smart default flow:
- Auto-detect URL column
- Auto-generate basic instructions from campaign
- Start a 10-job test run immediately
- Show live progress as it happens

### 2. One-Page Workspace
**Everything happens in the batch dashboard** - no navigation required:
- Configuration (GT columns, instructions, sampling)
- Execution control (start, pause, stop, adjust concurrency)
- Live monitoring (running tasks, progress, health)
- Results viewing (pass/fail, accuracy, exports)
- Iteration (edit instructions, re-run failed tasks)

### 3. State-Aware Transformation
**The dashboard adapts to batch lifecycle** - same page, different modes:
- **Setup Mode**: First time, no runs yet
- **Running Mode**: Tasks are executing
- **Paused Mode**: Execution paused
- **Completed Mode**: Run finished, results ready
- **Idle Mode**: Between runs, ready to execute again
